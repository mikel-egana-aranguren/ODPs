NAME: Cyclic Structure.

CLASSIFICATION: Domain Modelling.

MOTIVATION: Cyclic structures are very common in biology, yet the key of a cycle, thus, tha fact that only a set of given nodes forms it, needs of closure axioms. This ODP shows how to use such closure axioms.

AIM: To model a cyclic structure with a given and un-changeable set of nodes.

STRUCTURE:

SAMPLE:

ELEMENTS: The elements are the nodes of the cycle (individuals), the class to which they belong, and the property linking them (functional).

IMPLEMENTATION: The first step is to create the nodes and link them in a cycle. Each node should be an instance of the main class. Add an enumeration of the nodes as an equivalent condition to the main class, and an onlysome restriction to itself along the linking property, so nearly complete closure is achieved (instances of other classes cannot be used in the cycle and new different instances cannot be added).

RESULT: A cycle is modelled with individuals. The cycle is closed in some ways (no instances of other classes can be added) but not in others (a member can be changed or a link deleted).
